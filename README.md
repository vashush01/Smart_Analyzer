# 📊 Smart Analyzer

A Python-based desktop application that analyzes Excel datasets through a simple graphical interface. The application allows users to load Excel files, explore data, visualize insights with charts, and generate useful statistical summaries without writing any code.

---

## 🚀 Features

- 📂 Import Excel (.xlsx) datasets
- 📊 Interactive data visualization
- 📈 Generate charts using Matplotlib
- 🔍 View statistical summaries
- 🧮 Automatic data analysis
- 🖥️ User-friendly Tkinter GUI
- ⚡ Fast and lightweight desktop application

---

## 🛠️ Tech Stack

- Python
- Tkinter
- Pandas
- Matplotlib
- OpenPyXL

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/vashush01/Smart-Analyzer.git
```

Move into the project folder

```bash
cd Smart-Analyzer
```

Install dependencies

```bash
pip install pandas matplotlib openpyxl
```

Run the application

```bash
python smart_analyzer.py
```

---

# 📖 How It Works

### Step 1

Run the application.

The Smart Analyzer GUI opens.

---

### Step 2

Click **Browse** and select an Excel (.xlsx) file.

---

### Step 3

The application loads the dataset automatically.

---

### Step 4

Analyze the dataset using the available options.

You can:

- View dataset preview
- Check missing values
- Explore descriptive statistics
- Generate charts
- Visualize trends
- Analyze numerical columns

---

### Step 5

The generated charts and analysis help users understand the dataset more effectively.
